# Getting Started

First, install dependicies

```bash
npm i
# or
pnpm i
```

## .env file example

```env
AUTH_SECRET=
AUTH_URL=
AUTH_URL_INTERNAL=
AUTH_GITHUB_ID=
AUTH_GITHUB_SECRET=
NEXT_PUBLIC_SANITY_PROJECT_ID=
NEXT_PUBLIC_SANITY_DATASET
```

Then, run the development server:

```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
